# drf_demo
a django rest framework demo

## 说明
1.test.html是前端测试页面,可以用浏览器打开直接调用该项目API

2.项目拉取下来,`pip install -r requirements.txt`后,`python manage.py runserver`直接启动即可!

3.访问地址: http://127.0.0.1:8000
